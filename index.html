Grouping

When writing html code, you group your code into groups. This can be helpful when, for example, color coding a certain part of your page. By grouping things you can access a group of your code without individualy changeing each line. If you look into the code, you can see that because I group this into a <div>, I can change just this section of the browser to a different color.

Other Comands

Here are some of the other tag that can be helpful and what they do:

  • <ul> and <li>

    • The <ul> tag will put your code into a list using buletin points. However, in order to put you code into a list, you have to use the <li> tag. It is in this tag where you wrtie what you want to be listed. Another tag that makes a list is the <ol> tag. Instead of making a list in bulletin points, the tag makes a list using numbers.

  • <ol>

    • The <ol> tag also makes a list, just like the <ul>, but instead of listing the content with buliten points, it lists the content with numbers.

  • <em>

    • The<em> tag makes what ever code inside of it italcized.

  • <br>

    • This tag is different then other tag because you don't need to close it. It is a "self closing tag" Whenever you type <br>, your code will return one line. Depending on how many times you do code <ul> will determind how many times you go to the next line.

  • <strong>

    • The <strong> will bold whatever is in between the start and end of the two tags.

  • <img> , <scr> , and <video>

    • The purpose of this tag is to add images on to your web page. Like the <br> tag, it closes it's self. Right before the last ">" you write <scr>, and set it to the URL of the image you want to upload.
    The only difference between <img> and <video> is that <video> needs a close tag (looks like this: </video>), unlike the <img> tag
